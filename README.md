# TOMMY (ESPHome External Component)

This is the ESPHome external component for TOMMY, allowing you to add Wi-Fi motion detection capabilities to your existing ESPHome devices.

## Getting Started

The ESPHome Integration Guide covers everything you need to know about adding TOMMY to your ESPHome devices, including supported ESP32 variants, required configuration settings, and troubleshooting tips.

**Get started:** [ESPHome Integration Guide](https://www.tommysense.com/docs/flashing-devices/esphome)  
**Learn more:** [Full Documentation](https://www.tommysense.com/docs)
